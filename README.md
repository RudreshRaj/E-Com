# Project Setup

## Requirements

Dependencies installed:

    Node.js (>=22.x)

## Getting Started

### 1. Clone the Repository

    git clone https://github.com/Keyur-Gondaliya/E-Com.git

### 2. Install Dependencies

    $ npm install

### 4. Run

Go to the root and run

    $ npm run dev

### 5. Ruuning Urls

    Frontend : http://localhost:3000/
